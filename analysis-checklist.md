# 賽前分析 Checklist + Watchlist（執行層）

**權威規則（硬）：** `.github/skills/defensive-betting-analysis/SKILL.md`  
**賽後帳本 / 批次 review：** `post-match-review-grok.md`（表 A/B、逐場賽果）  
**本檔角色：** 賽前 **執行 checklist + watchlist**；與 skill 衝突時 **以 skill 為準**。

> **目的：** 避免「合理 PASS 變無腦全 PASS」——在 **穩定聯賽 + 清晰 WR 路徑** 把 Low lean **升一格**成小注正式 PLAY。  
> **防漏升（2026-07 帳本審查）：** 批次結束必掃 **clear 聯賽 -0.25**、**結構下盤 +0.25 / +1**、**平手外圍 lean** —— 達標卻只寫 PASS/lean = **執行過嚴**（見 §1d、§4）。  
> **不**改 skill 硬規則；**不**為「每卡至少一腳」硬找注；**不**放寬 light / NPL / 薄 +0.75。  
> cover-after-PASS **永不**回寫。  
> **07-23 暫時賽果補強：** Houston 正式和殺 -0.75 **L**；light -0.25 連殺；Nashville **-1 走**；深讓和殺/走盤再證 — 見 §1b、§2、§6。

**維護：** 改執行規則 / watchlist **只改本檔**；賽後樣本 append **只改** `post-match-review-grok.md`。

## 雙軌預設（賽程+賠率必做）

每當用戶輸入 **含賠率的賽程／多場盤**（除非明說只要一邊）：

| 軌道 | Skill | 輸出 |
|------|--------|------|
| **A. Defensive** | `defensive-betting-analysis` + 本 checklist | 最佳正式 **PLAY**（可多腳小注）、WR lean only、其餘 PASS；**含基本面修飾** |
| **B. Streak-roll** | `streak-roll-eval` | 最佳下一腳 **STREAK_LEG** 或 **none**（目標 5 連勝全權益滾；極嚴；**賠率+路徑+基本面**） |

編排權威：`.github/skills/match-card-dual/SKILL.md`（回應骨架 A/B/C）。

**禁止：** 為填滿 B 而硬給 STREAK_LEG；為填滿 A 而硬升 PLAY；**只寫賠率不寫基本面**。  
**允許：** 兩邊皆空（尤其 streak 常為 none）。

**基本面（兩軌都要）：** 動機／賽季階段、主力缺陣、主客與旅途、賽程密度、輪換／B 隊、風格是否偏和。  
- Defensive：基本面 **修飾** WR，可否決升格。  
- Streak：基本面 **可強制 NOT_ELIGIBLE**（輪換／無慾／結構缺陣），即使 1X2 好看。

## 0. 每場固定順序（先做完再決策）

1. [ ] **身份鎖**（隊名、主客、聯賽 vs 盃/跨國/首回、腿次）  
2. [ ] **外圍 1X2/AH**（命名 Sources Used；無外圍 → incomplete → 傾向 PASS）  
3. [ ] **力度分檔**：Strong / Medium / Light / 近均無 lean  
4. [ ] **Track B 路徑**：該盤哪些比分是 W / L / P（**-0.75 必須點明：和 = L**）  
5. [ ] **基本面快照**：動機、人員、主客/旅途、賽程、輪換；與 lean **同向或否決**  
6. [ ] **分支標籤** + self cross-check（可比樣本見 `post-match-review-grok.md` 表 A/B；低相似不硬 PLAY）  
7. [ ] **決策分層**（三選一，須寫明）：  
   - **正式 PLAY**（進表 A / 可下注）  
   - **WR lean only**（方向；**不**下注）  
   - **PASS**（無 usable edge 或 hard/ soft 擋下）

## 1. 正式 PLAY 升格門檻（須**全部**勾選）

僅當下列 **全部** 成立，才允許 **Confidence Low（或 Low–Medium）正式 PLAY · 小注**：

1. [ ] **穩定國內聯賽**（或同等結構）：非歐戰資格首回、非友誼/CLB/季前、非 NPL/澳盃跨級、非決賽 light  
2. [ ] 外圍 gap **≥ Medium**（主/客勝大約 **≤1.70**，或同等清晰客熱；**不是** ~1.90–2.10 light）  
3. [ ] 可執行盤 **Track B 友好**，例如：  
   - 熱門 **-0.75**（勝 1 = W），或  
   - 熱門 **-0.25** 且 lean 為 **clear**（約 ≤1.70 檔；**light 不升格**），或  
   - **結構下盤**（見 §1d）：穩聯賽 **dog +0.25**（和=W）或 **dog +1**（輸1=P），且外圍 **非強主 / 略偏 dog 側**，或  
   - **平手 + 外圍 clear lean**（牙山型；真五五仍 PASS），或  
   - 深讓僅當 **Strong** 且球數門檻可達（Hammarby 型；**罕見**）  
4. [ ] 分支 cross-check：**非**淨殺歷史；NPL / light-0.25 / 友誼 / 半職淺 dog **不**借歐聯命中洗白  
5. [ ] **基本面不否決 WR 案**（見下 §1-fund）：動機對齊、無結構性輪換/B 隊、無雙脊梁缺陣、旅途/賽程不致摧毀熱門；**輸出須有基本面快照**（非只寫賠率）  
6. [ ] 輸出寫清：`Decision: PLAY` + WR case + **Fundamentals** + Sources + Confidence Low + 小注  

**§1-fund 基本面否決（任一 → 不升格 / 降回 lean only 或 PASS）：**

| 否決 | 例 |
|------|-----|
| 輪換 / B 隊 / 明確休戰動機 | 盃優先、賽程擠塞主動輪換 |
| 無慾 / dead rubber | 榜尾無保級、已提前奪冠無動力 |
| 結構缺陣 | 主力前鋒+中軸或門將等 **雙脊梁** 缺；或單核心缺明顯改球路 |
| 旅途/賽程稅過重 | 客熱長途/三賽七天仍打深讓 |
| 未知主導 | 完全查不到人員/動機 → incomplete，**不**正式 PLAY |
| **-0.75 且明顯和局磁鐵** | 近交手連和、雙方偏不敗腳法，又無強取勝劇本 → 和=L 路徑放大，**傾向 lean only / PASS** |

**修飾（不翻邊，只調信心）：** 主場真實優勢、中段賽季穩定陣容、動機與 market lean 同向 → 支持 Low 小注；**不可**用基本面發明與外圍相反的 PLAY。

**範型（可升格）：**  
- 上盤：Lillestrøm / Hacken 型 — 穩定聯賽 + Medium + **-0.75** + 基本面無否決  
- 平半：穩定聯賽 + **clear ≤1.70** + **-0.25** + 基本面支持不敗  
- 下盤：艾比安 / 隆德里納型 — 穩聯賽 **+0.25** + 外圍非強主；或 **+1** 保護（比利時/半島型）  
- 平手 lean：牙山型 — 外圍 lean + 平手  

**不要升格成「只寫 lean」：** 上表全勾卻只寫 WR lean = **執行過嚴 / 漏升**。  
**不要升格成「只寫賠率」：** 缺基本面快照 = **未完成 §1**。  
**不要習慣性只找上盤半一：** 同卡若有 §1d 達標場，**必須**升或明示為何否決（不可默默 PASS）。

## 1d. 防漏升（批次必掃 · 帳本審查後）

> 目的：補回「好結構淺盤 / 下盤」被習慣 PASS 的漏；**不**放寬 light、NPL、薄 +0.75、歐戰首回。  
> 觸發：賽果帳顯示 early dog / clear 聯賽 shallow 有 WR 友好，但近期正式幾乎只剩 Medium **-0.75**。

### 1d-1 · 三條必掃（任一「全勾卻 PASS/lean」→ **改回正式 PLAY Low**）

| # | 結構 | 全勾條件（摘要） | Track B 友好點 | **不可**混入 |
|---|------|------------------|----------------|--------------|
| **A** | **clear 聯賽平半** | 穩國內聯賽 + 外圍熱門勝 **≤1.70** + 可執行 **-0.25** + 基本面不否決 | **和 = W** | 主勝≥1.85；歐戰首回；NPL；跨國 light |
| **B** | **結構下盤 +0.25** | 穩國內聯賽（**非** NPL/澳半職）+ 外圍 **主不強 / 略偏 dog 側或近均偏弱主** + 可執行 **dog +0.25** + 基本面不否決 | **和 = W**（半贏） | 羅奇類半職；外圍仍 **強主 ≤1.55** 硬撐 dog；動機送死 |
| **C** | **結構下盤 +1** 或 **平手 lean** | **+1**：明顯弱側、輸1可走、非五五亂鬥；**平手**：外圍 **clear lean**（非真五五）+ 基本面 OK | **+1 輸1=P**；**平手 lean 跟外圍** | 真五五平手；薄 **+0.75** 當 +1 替代（輸1=L） |

### 1d-2 · 下盤敘事（正式時必寫）

- **+0.25：**「和 = 半贏 = W；負才 L」+ 為何外圍支持護盤側  
- **+1：**「輸 1 = 走/P；本注保護」  
- **平手 lean：** 命名外圍 lean 側 + 「真五五才 PASS，本場有 lean」  
- 一律 **Low 小注**；**不**與同分支上盤熱門串關  

### 1d-3 · 仍強制 PASS（防漏 ≠ 亂開）

- dog **+0.75**（輸1=L）— Cincy/SKC cover **不**升格  
- NPL / USL / 澳盃 薄 dog  
- light 上盤 -0.25（≥1.85）  
- 僅因「本卡沒下盤」硬找受讓  

### 1d-4 · 批次輸出建議（可選短表）

```markdown
### 防漏升掃描
| 結構 | 本卡候選 | 結果 |
|------|----------|------|
| clear 聯賽 -0.25 | … / 無 | PLAY / 否決原因 / 無候選 |
| 結構 dog +0.25 | … / 無 | … |
| 結構 dog +1 或 平手 lean | … / 無 | … |
```

無候選時寫 **「防漏：無達標」** 即可（空是合法）。
