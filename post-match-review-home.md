# Post-Match Review Home

本檔只收：**賽後應注意點 / 分支提醒**，供之後賽前篩盤引用。  
不維護正式 PLAY 命中表、不 append 常駐帳本 A/B（那些仍以既有 review 檔為準）。

結算約定（與 skill 一致）：**贏半 = 贏，輸半 = 輸**；走盤不進 WR 分母。

**調整門檻（與 skill 對齊，勿單場改規則）：**
- `1` 結果 → Observation，不改 skill 硬規則  
- `2` 同類 → Watchlist，仍不改硬規則  
- `3` 同分支同類 → Formal review  
- `5`+ 可比樣本 → 才考慮 local rule tweak

3. **盤口對稱** — 讓/受近對稱 = 優勢極薄；對稱 + light → PASS 或 Low。  
4. **分支勿混** — 本場 L 只警告 light+高噪，不推翻乾淨聯賽 -0.25。  
5. **n=1 紀律** — 不反推「應買客」。

## 真五五 / coin-flip 黑名單 — 往績審視（外圍為主）

### 結論（策略）

| 說法 | 是否合理 |
|------|----------|
| **外圍無可用 lean → PASS**（無 WR edge） | **合理**，與 WR-first 一致，保留 |
| **圖上 AH 對稱（如 1.88/1.88）= 真五五黑名單** | **不合理（過粗）**；違反「外圍為主」，且往績已證過嚴 |
| **平手/近均 + 外圍有 lean → 可評估 lean 邊** | **合理**（牙山模型 / level-ball-lean） |

### Skill 原文本意（已對齊）

- 真五五定義應為：**外圍 1X2/AH 對稱或無 lean**，不是「可執行盤兩邊同價」。  
- 平手細分：真五五 PASS；**外圍 lean + 平手/淺盤 → 可跟 lean 邊**（confidence ≤ Medium）。  
- Soft gate：coin-flip 不可壓過 **已成立的外圍 WR edge**。  
- Hard gate 僅身份/錯場/無盤 — **不含**「對稱 AH」。

### 往績樣本

| 場次 | 事前誤用/用法 | 外圍 | 結果 | 含義 |
|------|----------------|------|------|------|
| **全南 vs 牙山** | Grok 見平手/近對稱 → coin-flip **PASS**（過嚴） | 略偏牙山 | 3-4 客勝；GPT **牙山平手 W** | **外圍 lean 被黑名單壓掉** → 必須改執行 |
| Wollongong 1.88/1.88 | PASS「真五五」 | 需外圍確認；圖對稱 | 2-3 | 任邊可開；若外圍真無 lean，PASS 合理 |
| AIK vs GAIS 近均 | PASS | 近均/略混 | 2-0 主 | 近均猜邊；PASS 不證「對稱必黑」 |
| 維京加高泰–史達蘭 | PASS 近均 | 近均 | 2-2 | 近均和；無 lean 則無 edge |
| 芝加哥–溫哥華 | 外圍 lean 平手 PLAY（pending） | lean 模型 | — | 與牙山同分支，勿用對稱黑名單否決 |

### 正確決策樹（強制）

```text
1) 先查外圍 1X2 / sharp AH（主）
2) 有 clear / medium lean？
   → 是：選 WR 路徑最好的可執行盤（可含平手 lean 邊）；圖價對稱只影響 execution/信心
   → 否：真五五 / no-lean → PASS（不是因為「兩邊都是 1.88」）
3) 僅有圖上 AH 對稱、外圍未查到？
   → Incomplete screen → PASS 或降級，標「外圍未確認」
   → 禁止寫「真五五黑名單」除非外圍已確認無 lean
```
